# Hardspace-Shipbreaker---Spolszczenie

Spolszczenie do wersji V.1.3.0.271424

Instrukcja instalacji spolszczenia
1. Plik "de" należy podmienić z plikiem w folderze Hardspace Shipbreaker\Data\LocDB;
2. Teraz wystarczy tylko w opcjach gry wybrać język polski.

Jeśli chcesz wersję bez polskich znaków, to należy pobrać plik "de_v2", zmienić jego nazwę na "de" oraz podmienić z oryginalnym plikiem.

Aktualnie przetłumaczone jest 88.35% tekstów, aczkolwiek tłumaczenie wymaga poprawek.


Korzystanie z nowszą wersją gry może powodowac zastąpienie brakujących napisów liczbami.
